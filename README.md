## 🛠️ Tech Stack

This portfolio is built with modern technologies to ensure high performance and SEO-friendliness:
- **Framework:** [Next.js 15](https://nextjs.org/) (React)
- **Styling:** [Bootstrap 5](https://getbootstrap.com/) & [Sass](https://sass-lang.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/) & [FontAwesome](https://fontawesome.com/)
- **Deployment:** Optimized for Cloudflare Pages / Vercel

## 🌟 Key Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Multi-language Support:** Available in English, Spanish, French, Korean, and Bengali.
- **Theme Customization:** Supports Dark, Light, Sepia, and custom color modes.
- **Interactive UI:** Smooth transitions, animated cursors, and engaging layouts.
- **SEO Optimized:** Built with semantic HTML and meta-tag management for better indexing.

## 📁 Project Structure

- `src/app`: Next.js App Router and page layouts.
- `src/components`: Reusable UI components.
- `src/providers`: Context providers for data, theme, and language management.
- `public/data`: JSON files containing all portfolio content (easy to update).
- `src/styles`: Global SCSS and theme-specific variables.
